## AUROC vs AUPRC — unified statement

Consistent with NeurIPS 2024 guidance (arXiv:2401.06091):

- We report AUROC as the primary metric for TCGA-KIRC (binary classification).
- We report C-index + Cox HR as primary metrics for IMmotion150 (survival analysis).
- AUPRC is reported as secondary for TCGA-KIRC (clinically interpretable at low prevalence).
- Binary AUROC on IMmotion150 is provided for completeness but explicitly noted
  as non-primary given the 62% event rate.

This approach is fully consistent with our pre-registered gate (which uses AUROC
for binary classification) and is honest about the appropriate use of each metric.
